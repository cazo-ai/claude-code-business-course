# Module 1: Getting Started - Exercises

Complete these exercises to reinforce your Module 1 skills.

---

## Exercise 1: File Reading Practice

**Objective:** Practice asking Claude to read different types of files.

**Tasks:**
1. Ask Claude to read the BRAND-VOICE.md file and explain FitHub's tone
2. Ask Claude to read the LOYALTY-PROGRAM.md file and summarize the tier structure
3. Ask Claude to read one month of customer feedback and identify the top complaint

**Success Criteria:**
- Claude successfully reads each file
- You receive clear summaries
- You understand how to request file reading in natural language

---

## Exercise 2: File Creation

**Objective:** Practice creating files with Claude's help.

**Tasks:**
1. Create a file called `first-impressions.md` in the student-files directory
2. In it, include:
   - Your initial thoughts about the FitRewards problem
   - One thing that surprises you about the data
   - One question you want to investigate further

**Success Criteria:**
- File is created in the correct location
- Content reflects your genuine observations
- File is properly formatted in markdown

---

## Exercise 3: Information Finding

**Objective:** Practice extracting specific information from files.

**Tasks:**
1. Find out how many total members FitHub has
2. Find out what the current monthly active user count is
3. Calculate the engagement rate (ask Claude to help)
4. Find out what the target engagement rate should be

**Success Criteria:**
- All numbers are correct (11,800 members, 1,950 active, 16.5% current, 35-40% target)
- You can explain where each number came from

---

## Exercise 4: Folder Exploration

**Objective:** Practice exploring folder structures.

**Tasks:**
1. Ask Claude to list all files in the `inherited-chaos/old-campaigns/` folder
2. Count how many campaign files exist
3. Ask Claude to read one campaign file and explain what the campaign was about

**Success Criteria:**
- You correctly identify 8 campaign files
- You can describe at least one campaign in detail

---

## Exercise 5: Question Formulation

**Objective:** Practice asking questions in natural language.

**Tasks:**
Ask Claude these questions (in your own words):
1. What files contain information about competitors?
2. What does the previous manager's notes say about the main problem?
3. What's the difference between FitRewards tiers?

**Success Criteria:**
- Claude understands your natural language questions
- You receive accurate, helpful answers
- You're comfortable asking questions without technical jargon
