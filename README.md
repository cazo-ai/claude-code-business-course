## Getting Started

### 1. Watch Module 0 Video
Start with the Module 0 setup video to install Claude Code and get your environment ready.

### 2. Open the Workspace
1. Open **Cursor** (or VS Code)
2. Go to **File > Open Folder**
3. Select the **workspace** folder from this package
4. Open terminal: `Ctrl+\`` (Windows) or `Cmd+\`` (Mac)

### 3. Launch Claude Code
In the terminal, type:
```bash
claude
```

### 4. Verify Your Setup
Run the setup verification:
```
start-module-0
```

This will confirm everything is working correctly.

### 5. Start Learning
Once setup is verified, begin Module 1:
```
start-module-1
```

## How to Use This Course

### Interactive Lessons
When you type `start-module-1` (or 2, 3, etc.), Claude becomes your instructor and guides you through hands-on exercises in the terminal.

### Lesson Scripts
The **lessons/** folder contains written scripts of what Claude will teach you. Use these to:
- Preview what's coming next
- Review concepts you learned
- Catch up if you miss something

### Exercises
The **exercises/** folder has practice problems for each module. Complete these to reinforce your learning.
